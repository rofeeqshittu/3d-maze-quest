Features

Real-Time 3D Maze Navigation

Experience a truly immersive environment with real-time raycasting and intuitive player controls. Each twist and turn brings a new challenge as you explore the labyrinth in first-person view.

Built with SDL2 & Raycasting

The game is built using SDL2 and C programming, with raycasting to simulate 3D visuals. With a smooth game loop and real-time calculations, enjoy responsive gameplay on a 2D grid.

Dynamic Player Movement

Seamless player controls make for an exciting experience. Navigate the maze with ease using keyboard input, optimized for precision and accuracy, creating a satisfying gaming adventure.
